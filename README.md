# Cinépolis Go Deluxe 🎬

Una experiencia cinematográfica premium diseñada con React y Tailwind CSS para una reserva de boletos y snacks fluida y elegante.

## ✨ Características
- **Cartelera Dinámica**: Explora los estrenos más recientes con una interfaz visualmente impactante.
- **Selección de Asientos Pro**: Interfaz interactiva para elegir la mejor ubicación en la sala.
- **Dulcería Premium (CandyBar)**: Agrega tus snacks favoritos directamente a tu orden.
- **Resumen en Tiempo Real**: Visualiza el desglose de tu compra en todo momento.
- **Diseño Ultra-Moderno**: Dark mode nativo con acentos en púrpura y efectos de cristalografía (glassmorphism).

## 🚀 Tecnologías
- [React 18](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/) (Estilizado y Layout)
- [Zustand](https://github.com/pmndrs/zustand) (Gestión de estado global)
- [Lucide React](https://lucide.dev/) (Iconografía)
- [Framer Motion](https://www.framer.com/motion/) (Animaciones fluidas)
- [Vite](https://vitejs.dev/) (Tooling de desarrollo)
